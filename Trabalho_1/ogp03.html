Redes Neurais Artificiais (RNA) e otimização¶

  • Uma rede neural artificial (RNA) uma tcnica de aprendizagem supervisionada.
  • A aprendizagem realizada durante um processo de otimizao.
    • Na literatura se diz que a RNA est "treinando" ou "aprendendo" quando se executa um algoritmo de otimizao.
    • Para "treinar" ou "ensinar" uma RNA basta resolver um problema de otimizao.
    • Este problema quer determinar uma funo que se ajusta a um conjunto de dados ( dataset ).
    • A soluo do problema de otimizao uma funo preditora que melhor se ajusta aos dados.
  • A grosso modo, so equivalentes as seguintes expresses:
    • "Treinamento/Aprendizagem" com dataset ,
    • Ajuste de funo ao dataset ,
    • Resolver um problema de otimizao.

Qual é a relação entre RNA e OGP?¶

  • As RNA profundas ( deep neural networks ) precisam resolver problemas de OGP sem restries para "aprender/treinar" com o dataset.
  • por isso que usaremos algoritmos de OGP para fazer o processo de "aprendizado" de RNA profundas.
  • Quanto mais profunda a RNA mais variveis de otimizao ter o problema associado ao "treinamento"

O que é um dataset ?¶

  • Um dataset um conjunto de dados geralmente organizado em uma tabela:
    0.1 0.7 ... 0.2 0
0.6 0.2 ... 0.9 1
0.8 0.1 ... 0.3 1
...
0.4 0.8 ... 0.1 0
  • Cada linha do dataset se chama observao, exemplo, amostra ou medida.
  • Uma linha comea com uma lista de atributos ( features ) e finaliza com uma ou mais respostas ( response , label ou target ).
    • No exemplo acima, na primeir linha se tem: 
        0.1 0.7 ... 0.2 0
      • Ela indica que os atributos so nmeros reais entre 0 e 1
      • Tambm indica que a resposta 0.
    • Na segunda linha: 
        0.6 0.2 ... 0.9 1
      • Ela indica que a resposta 1.

O que é um dataset ?¶

  • O nmero de atributos e respostas de um dataset geralmente fixo.
    • Notao: emprega-se $n$ para especificar o nmero de atributos.
    • Notao: emprega-se $m$ ou $N$ para especificar o nmero de linhas (nmero de observaes, exemplos, ..., medidas).
    • No exemplo no se pode saber quanto $n$ ou $m$ por causa das reticncias.
      • O dataset pode ser representado por uma matriz com duas partes $$ \begin{array}{c} dataset=[X\ \ Y]\in\mathbb{R}^{m\times(n+1)}\\ X\in\mathbb{R}^{m\times n}\\ Y\in\mathbb{R}^{m\times 1}\\ \end{array} $$
        • $X$ denominada matriz de atributos
        • $Y$ denominada matriz de respostas (no exemplo um vetor de $m$ entradas)

O que significa um dataset ?¶

  • Uma linha do datset a codificao de uma informao.
    • Por exemplo: uma foto tem 256 pixels e nela se tem um gato ou um cachorro.
    • Um dataset de gatos e cachorros poderia caber em um arquivo de $m=1000$ linhas:
      • cada linha tem $n=256$ atritutos e no final um $0$ (gato) ou um $1$ (cachorro).

Como graficar o dataset no Python¶

  • Poderemos ver na tela do computador datasets com duas ou trs colunas.
    • Se cada linha do dataset tiver um atributo $a_1$ e uma resposta $r_1$, ento
      • graficamos o ponto $(a_1,r_1)$ no plano (usaremos matplotlib.pyplot.plot).
    • Se cada linha do dataset tiver dois atributos $a_1$ e $a_2$ e uma resposta $r_1$, ento
      • graficamos o ponto $(a_1,a_2,r_1)$ no espao (plot3D).
  • Vamos comear por graficar um dataset com um atributo e uma resposta.

from numpy import *

def carregaDataSet(nomeArquivo):
    arquivo = open(nomeArquivo, 'r')
    linhas = arquivo.readlines() 
    m=len(linhas) # nmero de linhas
    n=len(linhas[0].strip().split()) # nmero de colunas
    n=n-1 # nmero de atributos
    print('\nDataset com m=',m,'exemplos e n=',n,'atributos.\n')
    X=zeros((m,n))
    Y=zeros((m,1))
    i=0
    for linha in linhas:
        linha = linha.strip().split() # retira '\n' e separa entradas
        linha = [float(i) for i in linha]
        X[i,0:n] = linha[0:n]
        Y[i,0] = linha[n]
        i += 1
    return X, Y

X, Y = carregaDataSet('dataset00.txt') # faz a leitura do arquivo e define o dataset

m,n=X.shape
if n==1:
    import matplotlib.pyplot as plt
    plt.plot(X,Y,'r.')
else:
    import matplotlib.pyplot as plt
    from mpl_toolkits import mplot3d
    ax = plt.axes(projection='3d')
    ax.plot3D(X[:,0],X[:,1],Y[:,0],'r.')

Como graficar funções escalares de uma variável no Python?¶

  • Seja $f:\mathbb{R}\rightarrow\mathbb{R}$ e um intervalo $[a,b]$.
  • Para graficar $f$ em $[a,b]$ no Python, temos que:
    • definir pontos $x$ o intervalo $[a,b]$ (linspace).
    • definir pontos $y$ correspondente a cada ponto $x$ ($y=f(x)$).
    • usar matplotlib.pyplot.plot

Como graficar funções escalares de duas variável no Python?¶

  • Seja $f:\mathbb{R}^2\rightarrow\mathbb{R}$ e dominio $\Omega=[a_x,b_x]\times[a_y,b_y]$.
  • Para graficar $f$ em $\Omega$ no Python existem muitas formas.
  • Aqui explicaremos como graficar pontos no espao:
    • definir pontos $x\in\Omega$ (pode vir de um dataset ).
    • definir valores de $z$ calculando $z=f(x)$.
    • usar plot3D
  • Recomendo executar o seguinte script no Spyder ou no PyCharm, para poder rotar a figura.

from numpy import linspace,zeros
from numpy.random import rand, randint
import matplotlib.pyplot as plt
from mpl_toolkits import mplot3d

ax=-10
bx=10
ay=ax
by=bx

def f(x):
    return -x[:,0]**2-x[:,1]**2

# inventando pontos em Omega
numPontos=1000
vrand1=[rand() for i in range(numPontos)]
vrand2=[rand() for i in range(numPontos)]
X=zeros((numPontos,2))
X[:,0]=randint(ax,bx,numPontos)*vrand1
X[:,1]=randint(ay,by,numPontos)*vrand2

Z=f(X)

ax = plt.axes(projection='3d')
ax.plot3D(X[:,0],X[:,1],Z,'r.')

Ajustando gráfico de funções aos dados¶

  • O desafio agora :
    • Conhecendo o dataset , como definir a funo que melhor se ajuste ao dados do dataset ?
  • Se conseguimos graficar o dataset , e reconhecemos que os pontos se "parecem" ao do grfico de uma funo "conhecida", podemos tentar modificar a funo conhecida para que se ajuste aos dados.
    • Exemplo 1: se os pontos do dataset esto todos prximos de uma "reta", ento vamos tentar definir a funo: $$ f(x)=ax+b $$ onde $a$ e $b$ devero ser ajustadas.
    • Exemplo 2: se os pontos do dataset esto todos prximos de uma "ondulao", ento vamos tentar definir a funo: $$ f(x)=a\ sin(bx+c) $$ onde $a$, $b$ e $c$ devero ser ajustados.
    • Exemplo 3: se os pontos do dataset esto todos prximos de um "sino", ento vamos tentar definir a funo: $$ f(x)=a\ e^{-a\ (x-b)^2} $$ onde $a$, $b$ devero ser ajustados.
